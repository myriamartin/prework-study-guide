HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git add -A: stage changes
  • git commit -m “your message here”: commit and add changes locally
  • git pull origin main: to make sure the local branch is up to date before pushing up the changes
  • git push: to push up your changes to the remote repository
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a programming language that allows users to interact with websites. One of its core features is variables, which store data types as values.
  • Use the var keyword to declare a variable. After the var keyword, use an assignment operator = to assign the specific data that we want stored in our variable.
  • We use the console.log() method to output a message to the web console by adding an argument. Developers use console.log method daily to debug and test the code that they write.
  • To connect the script.js file to the index.html file we add the script file under the footer element and above the closing element for body
  • Control flow: is the order in which a computer executes code in a file. Typically, JavaScript code is executed from top to bottom. However, using conditional statements such as if statements allows us to interrupt control flow.
  • An if statement is a way of introducing decision-making into our code. When using the if statement, code will only execute if the statement meets a condition, or is true. We can have more than one condition by using else if.
  • An array is a single variable that is used to hold a group of data that is related in some way. Arrays are useful tools for storing multiple items with different data types (boolean, string, numbers) in a single variable, but not all arrays must have multiple data types.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over, for loop that can iterate over the array and log each topic in the console with only a few lines of code.
  • A function is simply a set of instructions that tells the computer how to perform a certain task. You must call a function by name in order to execute it.
